# Sree S — Portfolio Website

A responsive personal portfolio for **Sree S**, Full Stack Python Developer, Data Analyst & Web Developer. Built with vanilla HTML, CSS, and JavaScript — no frameworks, no build step, just open and go.



---

## ✨ Features & Effects

| Effect | Description |
|---|---|
| 🎯 **Custom Cursor** | A dot + trailing ring cursor that smoothly follows the mouse and expands on hover over links, buttons, and cards |
| 🌌 **Particle Background** | A lightweight `<canvas>`-based particle field that drifts across the hero section, recoloring itself for light/dark mode |
| ⌨️ **Typing Animation** | Hero subtitle cycles through roles (*Data Analyst, Python Developer, Django Specialist...*) with a type/delete effect |
| 🃏 **3D Tilt Cards** | Service and project cards tilt in 3D based on cursor position, with a glow that follows the pointer |
| 🧲 **Magnetic Buttons** | Buttons subtly follow the cursor as it approaches, snapping back on mouse leave |
| 🔢 **Animated Counters** | Stats count up from 0 to their target value when scrolled into view (`IntersectionObserver`) |
| 🌗 **Light/Dark Theme Toggle** | Persistent theme switch (saved to `localStorage`) available in both the navbar and mobile sidebar |
| 🎬 **Scroll Reveal Animations** | Sections fade/slide into view as the user scrolls, with staggered delays |
| 📱 **Mobile Sidebar Navigation** | Slide-in sidebar menu with overlay, active-link highlighting, and synced theme toggle |
| 🧭 **Active Nav Highlighting** | Navbar and sidebar links highlight the current section as you scroll |

---

## 🛠️ Built With

- **HTML5 / CSS3** — semantic structure, custom properties for theming
- **Vanilla JavaScript** — all interactivity, no external JS libraries
- **[Remix Icon](https://remixicon.com/)** — icon set (via CDN)
- **Canvas API** — particle animation
- **Intersection Observer API** — scroll reveals & counters
