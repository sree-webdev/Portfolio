## ✨ Features & Effects

<table>
<tr>
<td width="50%" valign="top">

**🎯 Custom Cursor**
Dot + trailing ring cursor that smoothly follows the mouse and expands on hover over interactive elements

**🌌 Particle Background**
Canvas-based particle field drifting across the hero, recoloring itself for light/dark mode

**⌨️ Typing Animation**
Hero subtitle cycles through roles with a type/delete effect

**🃏 3D Tilt Cards**
Service & project cards tilt in 3D based on cursor position with a pointer-tracking glow

**🧲 Magnetic Buttons**
Buttons drift toward the cursor as it approaches, snapping back on leave

</td>
<td width="50%" valign="top">

**🔢 Animated Counters**
Stats count up from 0 on scroll via `IntersectionObserver`

**🌗 Light/Dark Theme Toggle**
Persistent theme switch saved to `localStorage`, in both navbar & mobile sidebar

**🎬 Scroll Reveal Animations**
Sections fade/slide into view with staggered delays as you scroll

**📱 Mobile Sidebar Navigation**
Slide-in menu with overlay, active-link highlighting, synced theme toggle

**🧭 Active Nav Highlighting**
Navbar and sidebar links highlight the current section while scrolling

</td>
</tr>
</table>

## 🚀 Getting Started

No build tools or dependencies required.

```bash
# Clone the repo
git clone https://github.com/sree-webdev/SreePortfolio.git
cd SreePortfolio

# Serve locally (pick one)
npx serve .
python -m http.server 8000
```

Then open `http://localhost:8000` — or just double-click `index.html`.
